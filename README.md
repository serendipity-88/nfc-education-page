# NFC 教育说明页 - 碰一下支付

两个教育页面，分别针对用户不同的顾虑：

## 页面列表

### 1. 碰一下安全感教育页 (`src/index.html`)
**目标：** 消除用户对安全的顾虑

**核心信息：**
- NFC 通信距离仅 4cm，无法远程盗刷
- 金融级安全保障
- 技术成熟，20 年历史

**互动：**
- 安全距离滑块（拖动查看 0-10cm 的安全范围）

**[预览链接](https://serendipity-88.github.io/nfc-education-page/src/index.html)**

---

### 2. NFC 电量教育页 (`pages/nfc-power-education.html`)
**目标：** 消除用户对耗电的顾虑，建议常开 NFC

**核心信息：**
- NFC 技术成熟，几乎不费电
- 建议常开 NFC，享受"碰一下"便捷体验
- 待机一天耗电不到 0.02%

**互动：**
- 实验 1：长按模拟碰一下（每次 0.001%）
- 实验 2：时长选择器（1h/2h/4h/8h/12h/24h）
- 实验 3：折腾这么久，省了多少？（选择每天开关次数，看一个月省的电够做什么生活场景）

**[预览链接](https://serendipity-88.github.io/nfc-education-page/pages/nfc-power-education.html)**
